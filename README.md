# Retail Sales Data Mining Project

## Overview

This project is a domain-specific data mining and dashboard development project based on a retail sales dataset.  
It follows the CRISP-DM methodology and aims to analyze customer behavior, product performance, and sales trends through data exploration, visualization, and dashboarding.

## Dataset

**Dataset Name:** Retail Sales Dataset

The dataset contains transaction-level retail sales records, including:

- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

## Project Objective

The main goal of this project is to explore and analyze retail sales data in order to:

- Understand customer purchasing behavior
- Identify high-performing product categories
- Analyze sales trends over time
- Support future customer segmentation and data mining tasks

## Business Problem

Retail businesses often struggle to fully understand customer behavior and product demand.  
Without proper analysis, it becomes difficult to make informed decisions about marketing, pricing, and inventory management.

This project helps address that problem by transforming raw sales data into meaningful insights and interactive visual analysis.

## Methodology

This project follows the **CRISP-DM** framework:

1. Business Understanding
2. Data Understanding
3. Data Preparation
4. Modeling
5. Evaluation
6. Deployment
